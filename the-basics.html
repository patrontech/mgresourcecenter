The BasicsLast updated: September 23rd, 2020

Virtual Meet & Greets allow customers to purchase tickets to have a one-on-one conversation with the celebrity of their choice. This will cover the basics of what a Virtual Meet and Greet consists of.

HostsLast updated: September 23rd, 2020

In every room there are a varying number of hosts which typically consist of: event staff members, the talent, the talent's agent, and a tech. The number of hosts and the host's titles can vary quite a bit but this example represents the most common situation for any given meet and greet. This guide will walk you through understanding the basics of how this works.

Hosts are uniquely identified in the room by using the host link that was created for them. Host links can be created in the Advanced Features section of a product.

Note

Attendees will show up on the list of hosts once they've been let into the room to begin their session. They will not have any controls that the rest of the hosts have. Two additional buttons are attached to their name labeled "Back to "Lobby" and "Finish Session".

Moderator

Personnel assigned by the event organizer to act as the primary supervisor for any given session. The moderators responsibilities include direct communication with the talent, ushering attendees into their session, starting the session timer, and adding additional time to a session in order to avoid an abrupt end to a conversation. If required, a moderator can be trained to run tech checks.

Talent

The Celebrity of the assigned one-on-one session. Please provide the reference materials from the Talent Resources page and instruct them to that content prior to their session.

Event Tech

An Event Tech determines if an attendee has their microphone, camera, and network connection configured properly prior to joining their session with the talent.

AreasLast updated: September 23rd, 2020

There are two "areas" that make up a Virtual Meet & Greet, which we'll outline below. Both the Room and Backstage serve a specific function.

Note

The only way a host in "The Room" can communicate with a host "Backstage" is by using the Host Chat.

Room

The area where Talent and Attendee have their one-on-one Meet & Greet session. All hosts, aside from the Event Tech, will stand by in the "Session Room" during the sessions to ensure it runs smoothly.

Backstage

If you have not joined the room but you have loaded the session then you are considered to be in backstage. Your name will appear on the list of hosts but you cannot communicate with them over audio/video unless you join the room. Backstage is where the assigned tech will reside for typically the entire meet and greet session. This is where tech checks will be performed. Hosts that are backstage can only communicate with hosts in the room via the host chat.
